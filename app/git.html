Guía de Git: Navegando en el Océano Digital

Paso 1: Verificar tu equipo de buceo (Git)

Comprueba que Git está listo en tu equipo.

git --version

Paso 2: Registrar identidad

Identifica tu expedición.

git config --global user.name "Tu Nombre"
git config --global user.email "tu.email@example.com"

Paso 3: Inicializar el arrecife

Activa tu carpeta como repositorio local.

git init

Paso 4: Capturar datos

Guarda tu primera versión.

git add .
git commit -m "Primer mensaje de expedición"

Paso 5: Crear repositorio en GitHub

  • Prepara tu repositorio remoto en GitHub.

Paso 6: Conectar a las corrientes

git branch -M main
git remote add origin https://github.com/tu_usuario/tu_repositorio.git
git push -u origin main
