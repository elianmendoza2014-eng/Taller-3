# Taller 3 - Maquetación Web con Bootstrap

Este proyecto es un portafolio web interactivo desarrollado por **Elián Mendoza** como parte de su formación en Análisis y Desarrollo de Software. La aplicación cuenta con un diseño moderno, responsivo y optimizado con temáticas inspiradas en el océano y degradados personalizados.

## Características

* **Diseño Responsivo:** Adaptado para dispositivos móviles, tablets y computadoras utilizando el sistema de rejilla (Grid) de Bootstrap 5.3.3.
* **Sección de Inicio:** Portafolio personal con datos curiosos y secciones interactivas.
* **Formularios Estilizados:** Interfaces limpias para el registro de usuarios e inicio de sesión con validaciones visuales.
* **Tablas de Datos:** Estructuras organizadas y responsivas para visualizar registros de proyectos.
* **Guía Interactiva de Git:** Un paso a paso visual e intuitivo para aprender los comandos esenciales de Git y GitHub bajo una temática marina.
* **Optimización SEO y Accesibilidad:** Estructura de encabezados ordenada y atributos aria-label implementados para cumplir con los estándares de Lighthouse.

## Tecnologías Utilizadas

* **HTML5** - Estructura semántica del sitio.
* **CSS3** - Estilos personalizados, degradados avanzados (gradients) y efectos visuales.
* **Bootstrap v5.3.3** - Framework de diseño para componentes y responsividad.
* **Bootstrap Icons v1.11.3** - Librería tipográfica para los iconos del sistema.
